# Getting Started

## Installation

<p style="font-size: 20px;">
To install NC², run the following command:
</p>

```sh
pip install nc2
```

<p style="font-size: 20px;">
After installation, you can run NC² from the command line as follows:
</p>

```sh
nc2 path/to/your/file.nc
```
